# Titanic Data Cleaning Report

## Problem Statement

The objective of this project was to clean and preprocess the Titanic dataset to make it suitable for analysis.

## Dataset

Titanic Dataset from Kaggle.

## Methodology

1. Loaded the dataset using Pandas.
2. Checked missing values.
3. Filled missing values in Age and Embarked columns.
4. Removed duplicate records.
5. Converted appropriate data types.
6. Renamed columns where required.
7. Exported the cleaned dataset.

## Results

The dataset was successfully cleaned and prepared for further analysis and machine learning tasks.

## Conclusion

Data preprocessing improved dataset quality and ensured that the data was ready for analysis.
