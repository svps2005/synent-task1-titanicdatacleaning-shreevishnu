Titanic Data Cleaning and Preprocessing

Problem Statement

The objective of this project is to clean and preprocess the Titanic dataset to make it suitable for analysis and machine learning.

 Dataset

Titanic Dataset from Kaggle.

 Technologies Used

* Python
* Pandas
* Jupyter Notebook

Steps Performed

* Loaded dataset
* Checked missing values
* Handled missing values
* Removed duplicate records
* Converted data types
* Renamed columns
* Exported cleaned dataset

 Results

The dataset was successfully cleaned and prepared for further analysis.
